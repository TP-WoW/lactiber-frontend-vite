# Troubleshooting — Índice

Guía de resolución de problemas para la aplicación, con diagnóstico, causas comunes e insights para solucionarlos.

Secciones:

- [Problemas del Backend y API](/troubleShooting/backend)
- [Problemas de Base de Datos](/troubleShooting/database)
- [Problemas Frontend y Build](/troubleShooting/frontend)
- [Problemas de Red y SSL](/troubleShooting/network)

Cómo usar esta guía:

- Sigue la sección que corresponda al síntoma (errores HTTP, timeouts, fallos en la UI, desconexiones).
- Revisa logs en frontend, backend y base de datos en el orden indicado.
- Anota reproducibilidad, timestamps y pasos exactos — facilitará la investigación.
